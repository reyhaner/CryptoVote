# 🛡️ CryptoVote - Güvenli Elektronik Oylama Sistemi

Bu proje, siber güvenlik prensipleri (CIA) doğrultusunda, seçmen gizliliğini ve oy bütünlüğünü korumayı hedefleyen bir simülasyondur.

## 🛠️ Kullanılan Teknolojiler
- **Dil:** Python 3.13
- **Kütüphaneler:** Tkinter, Cryptography, JSON
- **Metodoloji:** Hash Zinciri (Blockchain) & RSA/AES Şifreleme
